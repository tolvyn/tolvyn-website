Common Questions

Does TOLVYN read my prompts?

No. TOLVYN only captures metadata — token counts, model used, cost, latency. Your prompt and response content never touches our storage.

What happens if TOLVYN goes down?

Nothing. Our SDK is fail-open by design — if TOLVYN is unreachable, your AI calls route directly to the provider. Your business never stops because of us.

How is this different from checking the OpenAI dashboard?

OpenAI shows your total bill. TOLVYN shows which team, which service, which feature caused it — with budgets that block spending before it happens and an audit trail that holds up to scrutiny.

How long does integration take?

Five minutes. Change one import line in Python or Node.js. Or set one environment variable. No YAML, no agents, no config files.
