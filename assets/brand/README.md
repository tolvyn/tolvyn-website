# TOLVYN Brand Assets

## Files

| File | Use case |
|---|---|
| `wordmark-mint.svg` | Default — dark backgrounds (homepage, app, docs) |
| `wordmark-dark.svg` | White or light backgrounds |
| `wordmark-white.svg` | Mint or colored backgrounds |
| `favicon.svg` | Browser tab, bookmarks (scales 16→512px) |
| `favicon-dark.svg` | Alternate favicon for dark-only contexts |
| `app-icon.svg` | Source for iOS/Android home screen icons (no corner radius) |
| `og-image.svg` | Social card for Twitter/X, LinkedIn, Slack previews |

## Colors

| Name | Hex |
|---|---|
| Mint green | `#00D9A3` |
| Dark navy | `#0B0F1A` |
| White | `#FFFFFF` |
| Body text on dark | `#E2E8F0` |
| Muted text | `#94A3B8` |

## Typography

- Font: Inter Black (weight 900)
- Wordmark is always ALL CAPS
- Letter spacing: +6

## Do not

- Do not stretch or distort the wordmark
- Do not use the wordmark in any color outside the three provided
- Do not place the mint wordmark on green or low-contrast backgrounds
- Do not add effects (shadows, gradients, outlines) — keep it flat
